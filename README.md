### 💪🏻 Installation

> `Requirements`
> NodeJS v18+
> NPM v9.6+

**Installiere alle nötigen dependencies!**
```sh
npm install
```

**Bearbeite die config.js!**
`./CONFIGS/config.json`

**Starte den Bot!**
```sh
node index.js
```
